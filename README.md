# Marin Profils — base Astro propre

Projet vitrine Astro 5 pour Cloudflare Pages.

## Déploiement Cloudflare Pages

- Framework preset : Astro
- Build command : `npm run build`
- Build output directory : `dist`
- Root directory : vide

Important : ce projet ne contient volontairement pas de `package-lock.json`, afin que Cloudflare installe les dépendances proprement depuis npm.

## Modifier le site

- Accueil : `src/pages/index.astro`
- Pages : `src/pages/`
- Composants : `src/components/`
- Style global : `src/styles/global.css`
- Images : `public/images/`
